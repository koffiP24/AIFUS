PROBLEM:
--------
User could not connect/login - "Connexion impossible - Erreur serveur"
The frontend was configured to use the remote production backend instead 
of the local development backend.

ROOT CAUSE:
-----------
frontend/.env had: VITE_API_BASE_URL=https://aifus.onrender.com/api
This pointed to the production server, not the local backend at localhost:5000

SOLUTION APPLIED:
-----------------

1. backend/.env (UPDATED)
   FRONTEND_URL=http://localhost:5173
   FRONTEND_URLS=http://localhost:5173,http://localhost:4173
   
   Note: Backend CORS already allows localhost (see app.js lines 70-75)

2. frontend/.env (UPDATED)
   VITE_API_BASE_URL=/api
   
   This uses Vite dev proxy (see vite.config.js) to forward /api/* to 
   http://localhost:5000 in development mode.

3. README.md (UPDATED)
   Added note explaining Vite proxy usage in development
